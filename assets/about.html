Our Project

This webserver project is part of the Computer Networks 2 course, designed to demonstrate the practical application of networking concepts.

The project showcases a simple web server built in C, handling different types of media and providing a platform to explore various networking protocols and techniques.

Developer

Goals

  • To understand the inner workings of a web server.
  • To manage various file types and serve them efficiently over a network.
  • To implement error handling and optimizations.
